## About

This is my personal portfolio website — a showcase of my work, skills, and background as a full stack developer with 2+ years of experience building scalable, research-driven web applications.

The site features a clean, component-based architecture with a responsive layout and interactive UI, built entirely with modern web technologies.

## Running Locally

```bash
# Clone the repo
git clone https://github.com/khushipatel-10/Portfolio-Website.git
cd Portfolio-Website

# Install dependencies
npm install

# Start the dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
