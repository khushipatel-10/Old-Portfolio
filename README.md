## 🚀 Running Locally

```bash
# Clone the repo
git clone https://github.com/khushipatel-10/Portfolio-Website.git
cd Portfolio-Website

# Install dependencies
npm install

# Fire it up
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and you're in. 🎉
